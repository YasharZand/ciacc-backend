## Installation

```bash
$ npm install
```
or 
```bash
$ yarn install
```

## Running the app

```bash
# development
$ npm run start
# or
$ yarn run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
